: div> O que é? A tag div é usada para criar divisões (ou blocos) na página HTML. Para que serve? Agrupa elementos para que possam ser organizados, estilizados com CSS ou manipulados com JavaScript. [ Principais tags utilizadas em linguagem de programação (HTML) html: estrutura principal do documento. head: contém informações como título, links de estilo, etc. body: onde fica o conteúdo visível da página. h1 até h6: títulos. p: parágrafo. a: link. img: imagem. ul, ol, li: listas. br: quebra de linha. input: campo de entrada de dados. button: botão. div e span: blocos de agrupamento. --- 3. Laços repetitivos (em JavaScript) for: executa um bloco de código um número fixo de vezes. for(let i = 0; i 5; i++) { console.log(i); // Mostra 0 até 4 } while: executa enquanto a condição for verdadeira. let i = 0; while(i 5) { console.log(i); i++; } do...while: executa ao menos uma vez, mesmo que a condição seja falsa depois. : O que é o console? É uma parte das ferramentas de desenvolvedor que existem nos navegadores (como Chrome, Firefox, Edge). Para que serve o console? 1. Testar códigos JavaScript Você pode escrever e testar comandos diretamente nele. 2. Ver mensagens do código Usando console.log("mensagem") no seu código, você consegue acompanhar o que está acontecendo. 3. Encontrar erros Quando algo dá errado na página, o console mostra mensagens de erro para ajudar a identificar o problema. 4. Depuração (debug) Desenvolvedores usam o console para entender o comportamento do código e resolver bugs. background-color é usado para definir a cor de fundo de um elemento HTML. font-size: Define o tamanho da fonte. text-align: Controla o alinhamento do texto (esquerda, direita, centralizado ou justificado). color: Define a cor do texto. text-transform: Controla a transformação das letras (maiúsculas, minúsculas, capitalização).

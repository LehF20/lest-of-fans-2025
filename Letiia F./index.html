O que é a tag <div>?
A tag div é usada para criar divisões (ou blocos) dentro da página HTML.

Para que serve?
Serve para agrupar elementos que podem ser organizados com CSS ou manipulados com JavaScript.

Principais tags HTML e suas funções:

  • html: estrutura principal do documento.
  • head: contém o título e links externos como CSS.
  • body: contém o conteúdo visível da página.
  • h1 até h6: títulos de diferentes tamanhos.
  • p: parágrafo de texto.
  • a: link.
  • img: imagem.
  • ul, ol, li: listas.
  • br: quebra de linha.
  • input: campo de entrada de dados.
  • label: rótulo que identifica o que o usuário deve preencher.
  • button: botão clicável.
  • div e span: blocos para organizar elementos.

O que é a tag <input>?
A tag input cria campos para o usuário digitar ou escolher informações, como texto, número, data, senha, etc.

Para que serve?
Serve para coletar dados do usuário em formulários, como nome, idade, e-mail, etc.

O que é a tag <label>?
A tag label é usada para criar um rótulo associado a um campo input.

Para que serve?
Ela melhora a acessibilidade e ajuda o usuário a entender o que deve ser preenchido naquele campo.

Laços repetitivos em JavaScript:

for (let i = 0; i < 5; i++) { console.log(i); }
Repete enquanto a condição for verdadeira.

let i = 0;
while (i < 5) { console.log(i); i++; }
Repete enquanto a condição for verdadeira.

do...while:
Executa pelo menos uma vez, mesmo que a condição seja falsa depois.

O que é o console?
Ferramenta dos navegadores usada para testar códigos JavaScript.

Para que serve o console?

  • Testar comandos JavaScript rapidamente.
  • Exibir mensagens com console.log().
  • Identificar erros no código.
  • Fazer depuração (debug).

Propriedades CSS importantes:

  • background-color: cor de fundo.
  • font-size: tamanho da fonte.
  • text-align: alinhamento do texto.
  • color: cor do texto.
  • text-transform: transforma o texto em maiúsculo, minúsculo, etc.

Responsividade:
É a capacidade do site se adaptar a diferentes tamanhos de tela (celular, tablet, computador).

Propriedade alt:
Descreve a imagem caso ela não possa ser exibida corretamente.
